## 一、任务分配总览

本项目以项目要求中的18个一级功能模块为主体范围。注册登录、停车场CRUD、订单管理、支付模拟、数据表维护等内容作为支撑子功能，不单独拆为一级模块。

## 二、详细任务清单

### 程晓洋任务 — 数据、运营与监管闭环

| 模块 | 前端页面 | 后端/API | 关键逻辑 |
|------|------|------|------|
| M2 停车场数据联网接入 | 接入配置、同步状态、离线告警 | 停车场接入、数据校验、同步日志 | 标准化空位数据，超时未更新标记延迟 |
| M7 停车行为大数据分析 | 周转率、时长分布、饱和度图表 | 统计聚合接口、报表导出 | 按区域、停车场、时间范围聚合 |
| M8 动态定价策略引擎 | 价格规则、调价记录、公示说明 | 调价规则计算、价格版本管理 | 高峰浮动不超过固定价格200% |
| M13 车场运营管理后台 | 运营概览、车位列表、收入统计 | 车场运营数据、远程开闸模拟 | 管理员权限隔离和操作日志 |
| M16 设备运维管理平台 | 设备台账、在线状态、故障记录 | 设备状态、运行日志、告警处理 | 摄像头、道闸、诱导屏、充电桩统一管理 |
| M17 城市停车监管平台 | 监管大屏、供需分析、投诉热力 | 汇总指标、区域对比、政策评估 | 监管端展示汇总数据并做脱敏 |

### 程子浩任务 — 车主停车路径与营销闭环

| 模块 | 前端页面 | 后端/API | 关键逻辑 |
|------|------|------|------|
| M3 城市停车一张图 | 停车场地图/列表、筛选、排序 | 停车场查询、区域热度、距离排序 | 空余车位、价格、距离综合推荐 |
| M4 停车诱导信息发布 | 诱导屏列表、内容编辑、发布记录 | 诱导策略、发布模拟、关联车场 | 满位车场不得继续诱导驶入 |
| M5 智能导航与车位预约 | 推荐方案、预约确认、倒计时 | 预约创建、取消、确认、超时释放 | 空闲 -> 锁定 -> 占用 -> 释放 |
| M6 车牌识别与无感支付 | 入出场模拟、订单详情、支付结果 | 车牌识别模拟、计费、支付状态 | 首15分钟免费，不足1小时按1小时 |
| M15 反向寻车导航 | 车牌输入、车位位置、步行路线 | 当前订单查询、车位定位 | 仅对进行中停车记录有效 |
| M18 商圈停车联合营销 | 活动配置、优惠券、核销记录 | 营销规则、停车券核销、效果统计 | 同一订单不可重复叠加同类优惠 |

### 丁梓钊任务 — 感知、共享、扩展与车主端

| 模块 | 前端页面 | 后端/API | 关键逻辑 |
|------|------|------|------|
| M1 路内停车位视觉检测 | 检测点位、车位状态、人工复核 | 模拟检测结果、复核记录 | 状态包括空闲、占用、违停、异常 |
| M9 共享停车管理平台 | 共享发布、时段设置、收益统计 | 共享车位、预约、防冲突 | 同一车位同一时段不得重复预约 |
| M10 充电车位智能管理 | 充电车位列表、充电状态、告警 | 充电桩状态、预约、异常占用 | 故障充电桩不可预约 |

## 四、数据库设计（第1周完成）

### 核心表清单

| 表名 | 用途 | 支撑模块 |
|------|------|------|
| users | 用户账号、角色、联系方式、密码哈希 | M13、M14、M17 |
| vehicles | 用户车辆与车牌信息 | M5、M6、M12、M15 |
| parking_lots | 停车场基础信息、区域、费率、接入状态 | M2、M3、M13 |
| parking_spots | 车位编号、类型、状态、最后检测时间 | M1、M3、M5、M10 |
| reservations | 预约、锁定、取消、确认、过期记录 | M5、M9、M10 |
| parking_orders | 入场、出场、计费、支付、订单状态 | M6、M7、M13 |
| shared_spots | 共享车位、共享时段、收益状态 | M9 |
| charging_piles | 充电桩状态、故障、占用关系 | M10、M16 |
| violations | 违停证据、审核、处理状态 | M1、M11、M17 |
| monthly_cards | 月卡申请、审批、续费、有效期 | M12、M13 |
| devices | 摄像头、道闸、诱导屏、充电桩等设备台账 | M1、M4、M10、M16 |
| guide_screens | 诱导屏位置、关联车场、发布内容 | M4 |
| pricing_rules | 动态定价规则、时段系数、调价记录 | M8 |
| marketing_campaigns | 商户活动、优惠规则、核销记录 | M18 |
| analytics_snapshots | 周转率、饱和度、热度等统计快照 | M7、M17 |

### 关键状态约束

```text
车位状态：free -> locked -> occupied -> free；disabled状态不可预约。
预约状态：active -> confirmed/cancelled/expired。
订单状态：parking -> completed/cancelled。
设备状态：online/offline/fault。
违停状态：pending_review -> confirmed/rejected -> processed。
```
